    Today Special

    欧蕾咖啡

    法国人是欧蕾咖啡的拥护者，你在法国的早餐里会看到肚子圆圆的欧蕾杯。依照个人喜好程度慢慢加入鲜奶，即可享用一杯清凉香醇的欧蕾咖啡了。

    摩卡咖啡

    巧克力向来是咖啡的好朋友，冰摩卡更将他们的关系升华，摩卡是一种古老的咖啡，是由意大利浓缩咖啡，巧克力酱和鲜奶油牛奶混合而成的。

    椰香咖啡

    椰香洋溢的冰咖啡，经椰子和咖啡的浓醇相结合，在其享用美味时，犹如置身在夏威夷群岛的浪漫海边，采用了椰奶糖浆最顺口好喝。
